Documentação Htapps

Htapps é um ambiente de desenvolvimento construído em C# e JavaScript que permite a construção de aplicações Windows desktop com funcionalidades integradas que permitem uma liberdade maior para diversos objetivos.

Obs: Esta documentação é pendente de atualização e atualmente serve para englobar as funcionalidades atuais do projeto.

1. Utilização

Atualmente o Htapps se encontra em fase de desenvolvimento, isso significa que ele ainda não tem uma ferramenta de linha de comando própria para gerar projetos novos. Para criar um novo projeto, basta usar como base a pasta "debugRelease" do repositório oficial. Para começar a desenvolver a aplicação, basta editar o conteúdo do arquivo "app.html", não é recomendado mexer no "index.html" pois ele contém código necessário para a execução correta da aplicação.

API de Scripting - window

(f) ● window.resizeScreen(x, y)

Altera o tamanho atual da tela não maximizada.

(f) ● window.setTitle(title?)

Altera o título atual exibido na aplicação.

(f) ● window.alert(text, title?)

Altera o título atual exibido na aplicação.

(f) ● window.console.[log/error](text)

Dispara um log ou um erro no console.

(array) ● window.console.output

Output final do console.

(f) ● window.setIcon(path)

Altera o ícone de exibição da aplicação.

(object) ● window.sessionStorage

Fornece opções para o gerenciamento do sessionStorage na aplicação. Documentação Ofical
Obs: Não apresenta suporte a operações diretas, necessário [set/get]Item().
