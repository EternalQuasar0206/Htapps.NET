Htapps Documentation

Htapps is a development environment built in C# and JavaScript that allows you to build applications Windows desktop with built-in functionality that allows greater freedom for multiple purposes.

Note: This documentation is pending update and currently serves to encompass current functionalities from the project.

1. Use

Currently Htapps is in the development phase, which means that it doesn't have a tool yet. command line to generate new projects. To create a new project, just use the folder as a base. "debugRelease" from the official repository. To start developing the application, just edit the file content "app.html", it is not recommended to mess with the "index.html" as it contains code necessary for the correct execution of the application.

Scripting API - window

(f) ● window.resizeScreen(x, y)

Changes the current size of the unmaximized screen.

(f) ● window.setTitle(title?)

Changes the current title displayed in the application.

(f) ● window.alert(text, title?)

Changes the current title displayed in the application.

(f) ● window.console.[log/error](text)

Triggers a log or an error on the console.

(array) ● window.console.output

Final output from the console.

(f) ● window.setIcon(path)

Change the application display icon.

(object) ● window.sessionStorage

Provides options for managing sessionStorage in the application. Official Documentation
Note: It does not support direct operations, required [set/get]Item().
